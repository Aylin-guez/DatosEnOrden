Que es DatosEnOrden

Una capa de investigacion publica sobre datos abiertos.

DatosEnOrden organiza registros publicos ya disponibles y los conecta en vistas entendibles para personas, periodistas, organizaciones civiles y equipos tecnicos. El objetivo es hacer mas simple seguir la evidencia, no reemplazar la verificacion humana.

Que conecta

ChileCompra, Lobby, Transparencia Activa y datos presupuestarios de muestra.

Como funciona

Normaliza registros, crea afirmaciones verificables, conserva evidencia y proyecta relaciones publicas navegables.

Que muestra

Perfiles de entidades, cronologias, conexiones entre fuentes, evidencia y explicaciones neutrales.

Estado actual del proyecto

Prototipo local funcional, preparado para una futura salida publica.

Disponible: Streamlit local, capa de servicios, prototipo Reflex e investigacion por entidad.

En preparacion: arquitectura de despliegue, dominio datosenorden.cl y documentacion de seguridad.

No desplegado: la base PostgreSQL local no esta expuesta publicamente.

Principios

Transparencia, trazabilidad y prudencia.

Evidencia primero

Cada conexion debe poder revisarse desde su fuente o evidencia original.

Lectura neutral

La plataforma muestra relaciones registradas; no interpreta intenciones.

Arquitectura responsable

La base de datos debe permanecer protegida detras de servicios controlados.

Transparencia y seguridad

DatosEnOrden muestra informacion publica.

  • No afirma delitos.
  • No genera acusaciones automaticas.
  • No sustituye la investigacion ni la revision legal.
  • Cada conexion debe verificarse en la evidencia original.
