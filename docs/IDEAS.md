# DatosEnOrden - Ideas

> El codigo construye el sistema. La documentacion conserva la memoria.

Ideas no son compromisos. Sirven para conservar memoria sin contaminar el roadmap inmediato.

## Producto

- Explorador ciudadano de presupuesto DIPRES.
- Navegador de relaciones entre organismo, orden de compra, proveedor y monto.
- Fichas publicas de entidad con fuente y evidencia.
- Historial de cambios por organismo o proveedor.

## Datos

- Integrar DIPRES como primera capa ciudadana.
- Integrar SERVEL con separacion estricta entre resultado y opinion.
- Integrar transparencia activa y solicitudes de acceso a informacion.

## Infraestructura futura

- Object storage para payloads raw cuando el volumen lo justifique.
- Busqueda textual sobre entidades y claims.
- Read model de grafo para consultas interactivas.
- Snapshots historicos firmados por hash.

## IA futura

- Asistente ciudadano con respuestas citadas.
- Abstencion obligatoria cuando no exista evidencia.
- No generar acusaciones ni rankings reputacionales.
