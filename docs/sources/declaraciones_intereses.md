# Declaraciones de Intereses

Status: prototype

Local role: declaration sample records that connect people, public roles, organizations, and companies.

Commands:

- Loader: `python scripts/load_declaraciones_intereses_sample.py`
- Summary: `python scripts/declaraciones_intereses_summary.py`

Notes:

- Uses `LOCAL_TEST_DATA` / `NOT_OFFICIAL_DATA`.
- Does not infer conduct, conflicts, risk, or wrongdoing.
- Represents declarations as source records, claims, and evidence without adding database schema.
