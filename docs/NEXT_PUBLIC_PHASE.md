# Next Public Phase

Lista de producto para preparar el lanzamiento publico.

## Fuentes Reales

- Integrar primeras fuentes reales verificadas.
- Documentar cobertura y limites de cada fuente.
- Mantener avisos claros cuando una fuente sea parcial.

## Dominio Y SSL

- Comprar o conectar dominio.
- Activar HTTPS.
- Probar rutas principales en desktop y mobile.

## Despliegue

- Publicar MVP en entorno estable.
- Verificar Home, Biblioteca, Expediente, Reportes, Seguimiento y Fuentes.
- Mantener datos demo claramente marcados mientras no haya datos oficiales reales.

## Newsletter

- Crear una lista de espera o boletin simple.
- Publicar novedades de reportes y documentos explicados.
- No prometer frecuencia antes de tener proceso editorial.

## Redes Sociales

- Abrir perfiles iniciales.
- Publicar avances del MVP.
- Compartir documentos explicados y reportes ciudadanos.
- Mantener tono neutral y basado en fuentes.

## Monitoreo

- Revisar disponibilidad del sitio.
- Registrar errores visibles.
- Revisar paginas vacias o enlaces rotos.

## Feedback Ciudadano

- Definir canal de contacto.
- Recoger dudas de navegacion.
- Recoger reportes de textos confusos.
- Priorizar mejoras que ayuden a entender evidencia y fuentes.
